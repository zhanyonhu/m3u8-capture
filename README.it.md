# IFD M3U8 Capture

Estensione per [Internet File Downloader](https://app.zhi200bbs.com/apps/internetfiledownloader). Find m3u8 links, parse playlists, send items to the desktop app.

## Prima di iniziare

1. Install the desktop app: https://app.zhi200bbs.com/apps/internetfiledownloader
2. Keep the browser helper enabled (port **26519**).

## Installazione — Google Chrome

1. Download or clone this repository.
2. Open `chrome://extensions` → enable Developer mode → Load unpacked → **`chromium`** folder.
3. Pin the extension.

## Installazione — Microsoft Edge

1. Download or clone this repository.
2. Open `edge://extensions` → enable Developer mode → Load unpacked → **`chromium`** folder.
3. Pin the extension.

## Installazione — Mozilla Firefox

1. Download or clone this repository.
2. Open `about:debugging#/runtime/this-firefox` → Load Temporary Add-on → **`firefox/manifest.json`**.
3. Reload after each Firefox restart.

## Utilizzo

1. Select text with links → right-click → **Capture m3u8 from selected links** → **Download**.
2. Right-click an m3u8 link → **Download this m3u8 with Internet File Downloader**.
3. Use the on-page panel when streams are detected.
4. Open **Settings** for bridge port and optional headers.
